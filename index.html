HTML

  • The head element contains information the webpage
  • The body element represents the visible content shown to the user
  • Closing tags are indicated by a slash

CSS

  • CSS is used to style HTML elements
  • Classes can be used to style particular items that you designate with that class
  • You can use hex codes or names to refer to colors
  • CSS files have to be linked to their HTML files

Git

  • Git is used for version control
  • "git status" checks what branch you are currently on
  • "git checkout -b (branch name)" switches to a new branch
  • "git checkout (branch name)" switches to an existing branch
  • "git add -A" stages changes, -A means all
  • "git commit -m (text input)" commits changes and includes a message (-m)
  • "git pull origin main" confirms that current branch is up do date (best practice to do before pushing)
  • "git push origin (branch name)" pushes up changes to GitHub

Javascript

  • Javascript is used to add interactivity to a website
